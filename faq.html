Frequently Asked Questions

How do I upload resources to the library?
To upload resources, you need to be logged in. Once logged in, go to the Libraries page and click the "Upload Resource" button. You can then select a file (JPG, JPEG, or PDF) and add a description before uploading.
What types of files can I upload?
Currently, we accept JPG, JPEG, and PDF files. These formats are ideal for sharing building plans, technical documents, and visual resources.
How do I view uploaded resources?
You can view resources by clicking the view icon on any resource card in the Libraries page. This will open the resource in a new page where you can view it in full size and download it if needed.
Is my account information secure?
Yes, we take security seriously. All passwords are hashed before storage, and we use secure session management to protect your account.
